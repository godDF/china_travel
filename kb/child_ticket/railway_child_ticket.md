# 适用年龄

随同成年人乘车的儿童，年满6周岁且未满14周岁的，应当购买儿童优惠票；年满14周岁的，应购买全价票。每一名持票成年人可以免费携带一名未满6周岁且不单独占用席位的儿童，超过一名时，超过人数应购买儿童优惠票。

# 购票与证件

儿童年龄按乘车日期计算。购票时应使用儿童本人的有效身份证件；儿童没有有效身份证件时，可按铁路现行规则使用同行成年人的有效身份证件信息申明免费乘车儿童。免费乘车儿童也应在购票时提前申明。

# 单独旅行

除需要乘坐旅客列车通勤上学的学生，以及铁路运输企业同意在旅途中监护的儿童外，未满14周岁的儿童应当随同成年人旅客旅行。

# 提醒

儿童优惠票不是按身高判断。具体车次、跨境线路和最新执行要求，应以铁路12306和车站公告为准。
