# 官方预约

故宫博物院未授权第三方代理门票和展览预约，应通过“故宫博物院”官方小程序办理。故宫不售当日票，门票和展览通常在参观前7日20:00开放预约；除法定节假日外，周一闭馆。

# 实名检票

所有观众，包括优惠票观众和陪护人，都需要实名预约。参观当日应携带预约使用的有效身份证件原件，并按预约的上午或下午时段检票。票种、日期、时段和参观人员通常不能直接改签，应按官方退订规则重新办理。

# 入口与路线

午门是观众参观入口，参观结束后从神武门或东华门离院，整体实行南向北单向参观。应结合天安门地区安检和步行路线预留到场时间。

# 安检和寄存

入院前须接受安全检查，禁止携带物品按故宫官方目录执行。大型行李可以按现场标准寄存，现金、电脑、首饰等贵重物品以及易燃、易爆、易碎、有异味物品和食品不属于普通寄存范围。

# 临时变化

重大活动、极端天气或文物保护需要可能导致临时闭馆、区域关闭或调整开放时间。参观前应再次查看故宫官网和小程序公告。
