# 适用条件

符合铁路规定的在校学生，家庭居住地和学校所在地不在同一城市，且学生证、学生优惠卡和优惠区间等信息符合要求时，可以购买学生优惠票。中小学生按铁路规定使用学校书面证明办理。

# 次数与区间

每学年可购买家庭居住地至院校或实习地点所在地之间四次单程学生优惠票。优惠次数、乘车区间和资质应以铁路12306当前核验结果为准。

# 资质核验和证件

学生每学年乘车前应按铁路要求完成优惠资质核验。乘车时应携带购票所使用的有效身份证件以及铁路要求核验的学生身份证明。只有学生证通常不能替代实名制购票所需的有效身份证件。

# 新生和毕业生

新生、毕业生可按铁路公布的证明材料和次数规则购买学生优惠票。规则可能调整，应在购票前通过铁路12306核对最新适用席别、折扣和证明要求。
