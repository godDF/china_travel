# 新生购票资格

入学新生可以凭录取通知书购买学生优惠票。新生当年开学前可购买一次从家庭居住地到学校所在地的单程学生优惠票。购票和乘车使用新生本人的有效身份证件。

# 线上购买步骤

新生应使用本人身份信息注册铁路12306账户，并将优惠类型设置为学生。可在铁路12306 App 的学生优惠资质核验专区填写姓名、证件号码、学校、家庭所在地和预计毕业年份等信息，提交后购买学生优惠票。

# 核验与进站

新生没有学生优惠卡，不能按普通在校生方式完成优惠卡资质核验。线上购票后，可携带本人有效身份证件和录取通知书到车站窗口办理核验，再刷身份证进站；如果没有完成线下核验，应携带录取通知书和购票使用的有效身份证件，通过人工检票通道进站乘车。

# 注意事项

录取通知书规则适用于入学新生的规定区间和次数，不等同于普通在校生每学年四次优惠。购票前应核对通知书、本人身份信息、家庭所在地和学校所在地，最终以铁路12306当前页面和车站核验为准。
