# 传统预约方式

需要提前预约的公共文化场馆和旅游景区应保留人工窗口、电话专线等传统预约方式，方便不熟悉智能手机操作的老年人现场或电话预约。

# 代为预约

预约系统应允许家人、朋友帮助老年人预约。老年人可以按场馆或景区规则，使用当日有效预约码、购票信息截图以及本人有效身份证件办理入场核验。

# 现场名额与现金服务

景区和文化场馆应为没有提前预约的老年人保留一定数量的免预约进入或现场购票名额，并保留现金购票方式。达到限流上限时，现场仍可能需要错峰或等待，不能理解为任何时间都保证立即入场。

# 有效证件与个人信息

老年人可以按照场所要求使用身份证、老年证等有效证件登记。具体预约名额、优惠票价和入场材料由各景区结合当地政策执行，出行前仍应查询目标景区最新公告。
