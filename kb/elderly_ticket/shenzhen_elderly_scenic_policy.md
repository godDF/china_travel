# 适用人群

年满60周岁的老年人，包括符合条件的港澳台居民，可以按照深圳市相关敬老优惠政策享受政府投资建设旅游景区的门票优待。

# 景区范围与凭证

符合条件的老年人可凭本人身份证或港澳台居民居住证，免费进入深圳市政府投资建设的旅游景区；深圳市公园对老年人免费开放。具体景区是否属于政策范围，应以景区和主管部门公布的信息为准。

# 社会投资景区

政策鼓励社会投资建设的旅游景区向老年人提供优惠或免费，但具体方案由景区自主制定，不能直接套用政府投资景区的免费规则。

# 出行前确认

免费或优惠通常针对门票，不当然包含景区内观光车、索道、游船、演出等项目。出行前应通过景区官方渠道核对预约要求、证件原件和二次消费项目。
