# 没有全国统一老人票价格

景区门票定价和老年人优惠由景区性质、地方政策和价格管理方式共同决定，并不存在所有景区通用的全国统一“老人票”年龄、折扣或免票标准。不同地区、景区以及索道、观光车等二次消费项目可能采用不同规则。

# 政府指导价景区

对实行政府指导价的景区，相关部门要求依法对老年人等特定群体落实门票价格减免优惠。各地可以结合实际扩大优惠群体或提高优惠幅度。

# 市场调节价景区

对实行市场调节价的景区，国家鼓励经营者为老年人提供优惠，但具体年龄、折扣、证件和适用日期由景区公布。因此不能只凭其他景区经验判断目标景区政策。

# 查询与核验

购票前应查看目标景区官方渠道，重点核对年龄界限、是否需要预约、有效身份证件、优惠是否只针对首道门票，以及索道、游船、观光车和专馆是否另行收费。
