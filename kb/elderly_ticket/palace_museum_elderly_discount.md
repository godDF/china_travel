# 大门票优惠

60周岁以上（含60周岁）老年人凭本人有效身份证件原件享受故宫博物院大门票半价优惠。官方页面列示旺季大门票优惠价30元、淡季20元；票价调整时应以官方预约页面为准。

# 专馆优惠

钟表馆、珍宝馆等需另行购票的专馆不包含在大门票内。60周岁以上老年人凭有效身份证件原件可按官方政策购买专馆优惠票，当前页面列示为5元每人。

# 预约与核验

享受优惠的观众也必须实名预约。参观时应携带预约使用的本人有效身份证件原件，从规定入口完成实名核验。陪护人不因陪同老人自动获得优惠，应按本人条件预约购票。

# 核对提示

故宫开放时间、预约时段、临时闭馆和票价可能变化，应通过“故宫博物院”官方小程序或官网查询，不应通过未经授权的第三方代订。
